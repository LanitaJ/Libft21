### Installation
In the root of your repo, run this command:

`git clone https://github.com/LanitaJ/Libft21.git libft `
### Usage
1.in your_program.c write `#include "libft/libft.h"`

2.cd libft

3.make

4.cd ..

5.`gcc your_program.c libft/libft.a`

6.Enjoy :)
